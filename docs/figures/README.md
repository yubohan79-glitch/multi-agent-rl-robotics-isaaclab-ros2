# Figure Assets

This directory keeps generated visual assets grouped by use:

- `paper/`: paper-style method and system figures.
- `rl/`: reinforcement-learning architecture, Sim2Real figures and result plots.

SVG files are editable sources. PNG files are exported previews for GitHub rendering.

The current repository direction is object-centric world-model + SAC Flow / PolicyFlow training for multi-agent robotics. The retained high-level algorithm note is `docs/rl_world_model_flow_policy_plan.md`.

Current IsaacLab media is intentionally limited to:

- `docs/media/最终回放_三视角同步拼接版.gif`
- `docs/media/large_scale_50v50_isaaclab_replay.mp4`
